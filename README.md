# 📝 Task Master - Premium MERN Stack To-Do App

**Task Master** is a full-stack web application designed to help users manage their daily tasks with a clean, modern, and "Clear" user interface. It features a secure authentication system, including email verification to ensure account validity.

---

## ✨ Key Features

* **Secure Authentication:** User registration and login powered by **JWT** (JSON Web Tokens).
* **Password Security:** All user passwords are encrypted using **BcryptJS** before being stored.
* **Email Verification:** Integration with **Nodemailer** to send 6-digit verification codes via Gmail.
* **Full CRUD Operations:** Create, Read, Update (Mark Complete), and Delete tasks.
* **Responsive UI:** A beautiful, modern design built with **Tailwind CSS**.
* **Real-time Feedback:** Axios-driven requests with instant UI updates.

---

## 🚀 Tech Stack

### Frontend
* **React.js** (v19)
* **Tailwind CSS** (v3) - For UI styling.
* **React Router Dom** - For seamless page navigation.
* **Axios** - For API communication.

### Backend
* **Node.js & Express.js** - Server-side logic.
* **MongoDB & Mongoose** - Database storage.
* **JWT & BcryptJS** - Security and authentication.
* **Nodemailer** - For sending automated verification emails.
